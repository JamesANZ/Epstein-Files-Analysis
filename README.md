# Epstein Files Analysis

Analysis of court records and DOJ disclosures related to the Epstein case.

## Contents

- `perpetrator_summary.html` - Main analysis page with summaries of key individuals
- `court-records/` - PDF files from court cases
- `images/` - Photos of individuals mentioned
- `analysis/` - JSON data file with extracted information

## Usage

Open `perpetrator_summary.html` in a web browser. The page includes:

- Summary of each individual and their involvement
- TLDR sections for quick reference
- Expandable details sections with timeline, legal cases, locations, and connections
- Links to source documents on GitHub

Click "Additional Details" on any individual to see expanded information including timeline, case numbers, locations mentioned, victim statistics, and key excerpts.

## Source

Data extracted from:
- Court records (Southern District of New York, Southern District of Florida)
- DOJ disclosures
- Public court filings from 2008-2020

## File Links

All PDF file links point to this GitHub repository. Ensure files are uploaded to match the directory structure referenced in the HTML.
